# Projeto Telecall

Site institucional desenvolvido para a empresa Telecall como projeto acadêmico,
abrangendo desde a criação de páginas estáticas até backend com PHP e banco de dados.

## Sobre o projeto
Este projeto foi desenvolvido inicialmente sem controle de versionamento.
Posteriormente, foi reorganizado e versionado utilizando Git e GitHub,
demonstrando evolução técnica e boas práticas de desenvolvimento.

## Estrutura do repositório
- pt1-html-css — Versão inicial com HTML e CSS
- pt2-estilização-javascript — Melhora visual e introdução a funcionalidades com JavaScript
- pt3-php-mysql — Backend com PHP e MySQL
- pt4-deploy-aws — Versão final preparada para deploy na AWS

## Tecnologias utilizadas
HTML, CSS, JavaScript, PHP, MySQL, Git, GitHub, AWS

## Objetivo
Projeto criado para fins educacionais e portfólio.
